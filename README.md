# qnimate

An animated colorful [Voronoi diagram](https://en.wikipedia.org/wiki/Voronoi_diagram) 
powered by [d3](https://github.com/d3/d3).

## ScreenShot

![1]

## Live demo

https://micooz.github.io/qnimate

## Try it!

Install via npm:

```shell
$ git clone https://github.com/micooz/qnimate
$ cd qnimate && npm i
```

Run in development:

```shell
$ npm start
```

Build and bundle:

```shell
$ npm run build:prod
```
